PROJECT TITLE:Knights tour
PURPOSE OF PROJECT:The knights tour is an ancient puzzle where the 
user has to place a knight piece on a chessboard 
and go around the board only touching each square once.
This program solves the puzzle for you and plays it in slow motion.
